# Vue 3 + TypeScript + Vite
# 下载依赖
```
npm  install
```

# 运行项目
```
npm run dev
```
# 项目目录结构
```
├── dist/                          // 打包文件
└── src/
    ├── api/                       // 接口请求目录
    ├── assets/                    // 静态资源目录
    ├── common/                    // 通用类库目录
    ├── components/                // 公共组件目录
    ├── mock/                      // mock数据存放
    ├── router/                    // 路由配置目录
    ├── store/                     // 状态管理目录
    ├── style/                     // 通用样式目录
    ├── utils/                     // 工具函数目录
    ├── views/                     // 页面组件目录
    ├── App.vue
    ├── main.ts                    // 入口
├── index.html
├── tsconfig.json                  // TS 配置文件
├── vite.config.ts                 // Vite 配置文件
└── package.json
```
